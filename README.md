FMSC Donation
================

# Requirements:
- Node.js v5.10.1
- NPM v3.8.6
- Gulp 3.9.0

# How to Install:

After installing the requirements, execute those commands on your command line (inside the project folder):
```
$ npm install
```

# How to Execute:

Inside the project folder:
```
$ npm start
```
After that, you can access [`http://localhost:3000`](http://localhost:3000).
